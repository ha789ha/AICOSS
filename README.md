# 2023-AICOSS

## 환경설정
1. Python >= 3.7
2. Git으로 Repository 클론
```shell
git clone https://github.com/ha789ha/2023-AICOSS
cd 2023-AICOSS
```
3. Python Requirements 설치
```shell
pip install -r requirements
```
4. main.py 실행
```shell
python main.py
```
